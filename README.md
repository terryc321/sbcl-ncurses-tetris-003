
# tetris

exploration of using sbcl foreign function interface on ncurses c library

## tetris keys

a : left
d : right
s : down
w : spin left
e : spin right

q : quit

no scoring

no end game detected

no high scores

no fancy graphics

start tetris use run.sh

## usage 

rlwrap ~/bin/sbcl --script main.lisp

hard coded path in curses.lisp
